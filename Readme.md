## Documentation

[Tailwindcss](https://tailwindcss.com/docs)

## Installation

- Install [Node.js](https://nodejs.org/en/download/)
- Clone repository
- Go to project directory
- Install dependency with npm

```bash
  npm install
  npm build
```

- Install live server

```bash
  npm install -g live-server
```

- Run project

```bash
  live-server public
```
